# Project Images

Add your project screenshots/images here.

## Recommended Image Specifications

- **Format:** JPG or PNG
- **Dimensions:** 1200x900px (4:3 aspect ratio)
- **File Size:** < 500KB for optimal loading
- **Quality:** High resolution but optimized for web

## How to Add Images

1. Take screenshots or create mockups of your projects
2. Optimize images using tools like:
   - [TinyPNG](https://tinypng.com/)
   - [Squoosh](https://squoosh.app/)
   - [ImageOptim](https://imageoptim.com/)
3. Save with the exact filenames listed above
4. Place them in this directory

## Placeholder Behavior

If an image is not found, the system will automatically display a themed icon placeholder matching the project category.

## Tips for Great Project Images

- Use real screenshots when possible
- Include mockups for mobile apps (use mockup generators)
- Show the UI/UX in action
- Ensure good contrast and readability
- Consider using tools like Figma or Sketch for polished mockups
